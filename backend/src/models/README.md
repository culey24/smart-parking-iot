# Models Module
Thuộc **Task 1**. 
Chứa các Mongoose Schemas (Database Entities).
Hãy tham khảo `docs/specs/class.md` và `docs/specs/class-1.md` để tạo các schemas:
1. `User`, `TemporaryCard`, `ParkingSession`, `Zone` (từ class.md).
2. `SystemConfig`: Lưu cấu hình động.
3. `PricingPolicy`: Lưu chính sách giá vé.
4. `AuditLog`: Lưu vết các thao tác admin.
5. `InfrastructureAlert`: Các cảnh báo lỗi hạ tầng (VD: sensor hỏng).
6. `ReconciliationRequest`: Yêu cầu đối soát với BKPay.
7. `Gate`: Cổng ra vào.
8. `IoTDevice`: Trạng thái các cảm biến và gateway.
