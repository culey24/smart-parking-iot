# Services Module
Nơi chứa toàn bộ **Logic nghiệp vụ lõi (Business Logic)**.
Các Service dự kiến:
- `AuthService` (Task 2): Gọi SSO, gen JWT token.
- `BillingService` (Task 4): Thuật toán tính phí dựa theo `PricingPolicy`.
- `ReconciliationService` (Task 4): Logic so khớp data giữa DB và ngân hàng.
- `NavigationService` (Task 6): Tìm kiếm khu vực còn chỗ trống.
- `ReportService` (Task 8): Tổng hợp doanh thu, hoạt động theo chu kỳ.
- `SystemAdminService` (Task 7): Logic validation khi đổi cấu hình hệ thống.
