# Controllers Module
Chứa các logic Http Request/Response. Phân chia theo các Task:
- `EntryExitController` (Task 3): Handle check-in/out, tra cứu thẻ, mở barrier.
- `PaymentController` (Task 4): Tính phí, lịch sử billing, logic đối soát.
- `IoTDataController` (Task 5): Webhook nhận data, list devices, alerts.
- `DashboardController` (Task 6): Số liệu thống kê màn hình chính, live monitoring.
- `SystemAdminController` (Task 7): CRUD SystemConfig, PricingPolicy, AuditLogs.
- `ReportController` (Task 8): API báo cáo doanh thu, hoạt động.
- `AuthController` (Task 2): API login, user profile, role.
